## 🚀 About The Project

A Python-based IP Geolocation and SSL Lookup tool that retrieves detailed information about any public IP address including:

- 🌐 Country, State & City
- 📍 Latitude & Longitude
- 🏢 Hosting Provider / ISP
- 🗺️ Google Maps Location
- 🔐 SSL Certificate Information
- 📅 SSL Validity Dates

Built to practice:
- Socket Programming
- API Integration
- SSL Handling
- Networking Concepts
- CLI Application Development
